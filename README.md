## About

"modreign" is a custom mod manager for ELDEN RING: NIGHTREIGN.

## Requirements

* ELDEN RING: NIGHTREIGN (Steam)

## Installation

Place the 'modreign' folder in your ELDEN RING: NIGHTREIGN game installation directory then run the following file:

```
install.bat
```
